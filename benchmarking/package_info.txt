MAFFT (for aligning sequences): 
    download: brew install mafft
    version: v7.526 (mafft --version)

    Example: mafft --auto data/test1.fasta > aligned_test1.fasta

    download: brew install hmmer
    version: 3.4 (Aug 2023), check through: hmmbuild -h
